void MultiBoost::ConstantLearner::getStateData ( vector< FeatureReal > &  data,
const string &  ,
InputData *  pData = 0 
)
virtual

Returns a vector of float holding any data that the specific weak learner can generate using the given input dataset. Right now just a single case is contemplated, therefore "reason" is not used. The returned vector of data correspond to:

\[ \left\{ v^{(t)}_1, v^{(t)}_2, \cdots, v^{(t)}_K, \phi^{(t)}({\bf x}_1), \phi^{(t)}({\bf x}_2), \cdots, \phi^{(t)}({\bf x}_n) \right\} \]

where \(v\) is the alignment vector, \(\phi\) is the discriminative function and \(t\) is the current iteration (to which this instantiation of weak learner belongs to).

Parameters
dataThe vector with the returned data.
pDataThe pointer to the data points (the examples) previously loaded.
Remarks
This particular method has been created for analysis purposes (research). If you need to get similar analytical information from this weak learner, add it to the function and uncomment the parameter "reason".

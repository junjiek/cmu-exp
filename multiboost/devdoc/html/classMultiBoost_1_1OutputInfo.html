Detailed Description

Format and output step-by-step information. With this class it is possible to output and update the error rates, margins and the edge. These function must be called at each iteration with the newly found weak hypothesis, but before the update of the weights.

Warning
Don't forget to begin the list of information printed with outputIteration(), and close it with a call to endLine()!

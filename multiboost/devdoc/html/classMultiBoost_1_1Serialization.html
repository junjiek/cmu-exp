Create a set of xml tags that describes a vector. For instance, if the vector values is [-1, 1, 1], with tagName="vArray", and the classes are ["one", "two", "three"] will return: 

<vArray size="3">
<class id="one">-1</class>
<class id="two">1</class>
<class id="three">1</class>
</vArray>

Default values are not saved (sparse representation)

Parameters
tagNameThe name of the tag.
valuesThe vector of the values.
nameMapThe map that contains the value<->is map.
elementTagNameE.g. "class" for the v vector.
defaultValueThe default value that is not explicitly saved.
numTabThe number of tabs before the tag. Useful for indentation.
